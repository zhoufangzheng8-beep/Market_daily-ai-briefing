## 3. 边际信息解读

**核心边际 1｜外盘科技股重挫是直接导火索（产业 + 资金面）**
6/25（美东）美股大型科技股集体杀跌：**苹果跌 5–6%**，并宣布因内存 / 存储芯片成本上行，MacBook、iPad **涨价约 20%**；英伟达、谷歌、微软、亚马逊、Meta 普跌，纳指承压（衔接昨日美股简报中"OpenAI IPO 或延期 + 美联储转鹰"的脆弱情绪）。叠加**日韩存储芯片龙头大跌**，全球科技风险偏好同步恶化，直接砸向 A 股 / 港股的**算力硬件、光通信、半导体**链条——这是今日两市普跌的外部底色。

**核心边际 2｜"成本端涨价"正在重塑科技股逻辑（产业）**
存储 / 内存涨价→终端（苹果）被动涨价→科技硬件**毛利与需求的双重担忧**。这一逻辑对 A 股的两面性正在分化：利空"纯算力 / 代工"环节（杀估值），却利好**上游涨价主线**（存储、玻纤、覆铜板、MLCC）。今日玻纤 / 玻璃基板逆势走强，正是资金在"涨价受益 + 先进封装"方向寻找避风港。

**核心边际 3｜缩量普跌 = 阶段性调整而非趋势反转（资金面）**
机构主流观点：本次下跌是**内外因素叠加的阶段性调整**，并非趋势反转；成交缩量、南向长线资金未撤、上证守住 4,000 点，均指向情绪修复空间。若**周末无新增外部冲击**，下周初存在技术性反弹窗口。

**核心边际 4｜政策面暂处真空，缺乏对冲（政策）**
当日无重磅央妈 / 监管 / 财政利好对冲外部利空，是普跌得以宣泄的原因之一。后续需关注**半年末流动性投放、7 月政治局会议预期**能否提供政策边际支撑。

## 4. 明日 / 短期关注点

**重要事件 / 数据**
- **6 月 PMI**（约 6/30 国家统计局发布）：验证内需与制造业景气，是下周初情绪的关键锚。
- **半年末资金面**：跨季流动性、央行公开市场操作力度。
- **7 月政治局会议**预期升温：下半年政策定调，是否有增量稳增长 / 资本市场表述。
- **外盘**：周末美股 / 费城半导体走向，及苹果涨价后的科技股情绪能否企稳。

**技术关键位**
- 上证 **4,000 点**为短线多空生命线，下方关注 3,950；上方反弹看 4,080–4,100。
- 创业板 4,200 缺口回补压力，企稳需重回 4,250 上方。
- 恒指 23,000 得而复失，关注能否快速收复。

**风险提示**
- ① 全球科技股若周末继续下杀，下周一两市或低开承压，算力 / 半导体链仍是高波动区。
- ② 存储涨价"成本端逻辑"持续，警惕科技硬件**杀毛利**预期发酵。
- ③ 政策真空期，缺乏对冲工具，普跌中需防个股流动性踩踏（如特发信息跌停式杀跌）。
- ④ 机会端：玻璃基板 / 玻纤 / 涨价主线若放量持续，或成调整中的新主线，但需防主题股高位回撤。
