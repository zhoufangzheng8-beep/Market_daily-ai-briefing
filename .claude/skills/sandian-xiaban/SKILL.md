# 三点下班 · 双视角对撞分析

把一只个股 / 一个主题，分别送进两位主播的对立体系里互相拷问，逼出买入理由的漏洞，得到一个更立体的判断。

> 完整方法论见 `podcasts/三点下班/投资方法论.md`；本 Skill 是其可执行版。

## 何时使用
- 用户给出一个标的或主题（如"分析一下铜""恒生科技现在能不能买""帮我用三点下班的框架看看 XX"）。
- 用户想要"买入逻辑 / 卖出信号 / 仓位 / 跟车风险"的结构化拆解。

## 执行步骤

### 第 0 步 · 归位（最重要）
先判定：**这笔投资赚的是"变化价值"的钱，还是"价值锚回归"的钱？** 明确写出来。两者买入理由、持有逻辑、卖出信号完全不同，不允许混用。若两者都沾，分别说明各占多少。

### 第 1 步 · 浩哥视角（变化价值 / 进攻）
用"十倍股四标准"逐条打分（✅/⚠️/❌ + 一句话理由）：
1. **超低估值**（进入时是否还没被充分定价）
2. **高速增长**（收入/利润/关键运营指标是否陡峭上行）
3. **巨大空间**（天花板能否容纳数倍想象）
4. **异类 outlier**（是否与众不同、共识尚未形成）

再补：产业拐点信号是什么？最可能让逻辑**证伪**的事件是什么？

### 第 2 步 · 星辰视角（价值锚 / 防守）
检查"价值锚三件套"：
1. **便宜锚**：估值是否低于横向（海外/同业）与纵向（自身历史周期底部）锚？
2. **向上期权**：有没有出海/复苏等业绩抬升的可能？
3. **向下保护**：分红 + 回购是否 ≥ ~5%，提供安全边际？

再补：这是真便宜还是**价值陷阱**（便宜是否因基本面持续恶化）？

### 第 3 步 · 共振检查
**个股逻辑 × 板块风口 × 大盘环境** 三者是否同向？逐项标注，并指出当前处于"冰"还是"火"的一侧。

### 第 4 步 · 仓位与卖出纪律
- **仓位建议**：真看懂→可重仓；龙头可稍重；非主流→轻仓；没机会→拿现金等。
- **卖出触发条件**（不用价格锚）：写明①逻辑破坏的具体信号 ②空间透支/估值修复到位的具体位置。

### 第 5 步 · 抄作业 / 跟车风险（若该标的是"跟来的"）
- 你抄的是**动机**还是只抄了**动作**？
- 有没有**交叉验证**（多信源）？
- 信息源是否只报喜（避雷）？
- 提醒：稀缺的是"判断"那一层，AI/研报已抹平信息与框架层。

### 第 6 步 · 心理体检（提醒用户）
- 现在的冲动是"怕踏空"还是"被套否认"？点名它。
- 若处于"上头"状态，建议先做 **54321 着陆法**（5 看 / 4 听 / 3 触 / 2 闻 / 1 尝）降温，再按纪律决策。

### 第 7 步 · 收束
给出**对撞结论**：浩哥会怎么说、星辰会怎么说、分歧点在哪、什么信号出现时谁对。给一个"如果是我"的折中判断，但强调由用户独立决策。

## 输出格式
用清晰小标题分块（归位 → 浩哥视角 → 星辰视角 → 共振 → 仓位与卖出 → 跟车风险 → 心理 → 对撞结论）。多用表格/清单。语气可带两位主播"互怼"的风格，但论点要扎实。

## 硬性要求
- **结尾必须附**：⚠️ 本分析为框架演练，不构成投资建议；市场有风险，请独立判断、自负盈亏。
- 数据/估值若用户未提供，明确标注为"需核实"，不要编造具体数字。
