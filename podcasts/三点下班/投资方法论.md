# 《三点下班》投资方法论（蒸馏版）

> 把节目里反复出现的框架，整理成一套可操作的方法论。核心是**两套对立体系 + 一套共识底层**。
> ⚠️ 学习性整理，非投资建议。

---

## 0. 总纲：一笔投资必须先归位到哪条主轴

每看一个标的/主题，先问：**我赚的是"变化价值"的钱，还是"价值锚回归"的钱？** 两者的买入理由、持有逻辑、卖出信号完全不同，混用是亏钱主因。

| | 浩哥派：变化价值 | 星辰派：价值锚 |
|---|---|---|
| **赚什么钱** | 产业革命带来的**预期/空间重估** | 错杀后的**均值回归 + 分红** |
| **买入前提** | 拐点信号、渗透率曲线启动 | 估值显著低于锚、有安全边际 |
| **最怕** | 逻辑证伪、产业证明是泡沫 | 价值陷阱（便宜是有原因的） |
| **持有依据** | 趋势未破、空间未兑现 | 估值未修复到合理 |
| **卖出** | 逻辑破坏 or 空间透支 | 回到合理估值 / 逻辑变坏 |

---

## 1. 浩哥体系：变化价值 / 产业革命（进攻）

**信条**："买股票就是买公司""选时代的公司"。赚的是产业从 0→1、1→10 的**变化**。

### 选股四标准（"十倍股巡礼")
1. **超低估值**——进入时还没被充分定价。
2. **高速增长**——收入/利润或关键运营指标陡峭上行。
3. **巨大空间**——天花板足够高，能容纳十倍想象。
4. **异类（outlier）**——与众不同、当下不被主流理解（共识未形成 = 还有钱可赚）。
   *案例：泡泡玛特。*

### 方向偏好（2025–2026）
游戏出海、AI 应用链、工业金属（**铜**）、精神消费、端侧芯片、Token 出海。

### 持有与心性
- 难点不在"发现"而在"**拿住**"——靠**认知**支撑，认知不够就拿不住十倍股。
- 养成"**赚大钱的心性**"：敢重仓自己真正看懂的产业拐点，容忍高波动。
- 个人投资者的优势：**相信直觉、时间优势、抗波动**；不和机构拼数据/模型；拉长周期看趋势。

---

## 2. 星辰体系：均值回归 / 价值锚（防守）

**信条**：用估值和安全边际说话，等市场把错杀的价格还回来。

### 买入四理由（以恒生科技为例的模板）
1. **估值低于海外同类**科技公司（横向锚）。
2. 处于**周期底部**（纵向锚）。
3. **出海**带来的业绩抬升（向上期权）。
4. **分红 + 回购 ≥ 5%**，提供**安全边际**（向下保护）。

> 通用化：一个"价值锚"标的最好同时具备 **便宜（横/纵向锚）+ 向上期权 + 向下保护** 三件套。

### 风险警觉
- 最大敌人是**价值陷阱**：便宜可能是基本面持续恶化的结果。
- 因此"低估值"必须配"逻辑没坏 + 有催化/回购托底"，单纯便宜不买。

## 3. 共识底层（两人都认同的部分）

### 3.1 流派与打法地图（S1）
- **流派**：企业流（看公司）/ 信息流（看边际变化）/ 技术流（看量价情绪）。
- **打法**：套利 / 周期 / 宏观。
- 选股要追求 **个股逻辑 × 板块风口 × 大盘环境的"共振"**，三者同向胜率最高。

### 3.2 卖出纪律
卖出由**触发条件**驱动，而非情绪：
- **逻辑破坏**（买入理由不再成立）→ 无条件卖。
- **空间透支 / 估值回到合理**（兑现完毕）→ 减或清。
- 不靠"涨了好多/亏了好多"这种**价格锚**做卖出决策。

### 3.3 仓位
- 有真正看懂的好机会 → 重点配置；没有 → **拿现金耐心等**，轻仓练盘感。
- 主流热点买**确定性龙头**，仓位可稍重；非主流只**轻仓参与**。
- 现金流与风险管理是底盘（S5E1 反复强调）。

## 4. 心理与执行系统（节目的灵魂）

### 4.1 情绪是头号风险
- **踏空 > 被套** 的痛苦量级：踏空引发抑郁/自我否定，被套触发否认机制。识别自己正处在哪种状态，是止损/止盈前的第一步。
- 警惕**投资信息成瘾**（刷盘=赌博式成瘾）；关注"**电量低**"的躯体信号。

### 4.2 54321 着陆法（交易"上头"时用）
当冲动要追/砍时，强制做一遍正念着陆，把自己从念头拉回当下、回到**流程**：
> 觉察 **5** 个看到的 → **4** 个听到的 → **3** 个触到的 → **2** 个闻到的 → **1** 个尝到的。
降温后再按纪律决策，而不是在情绪峰值下单。

### 4.3 漩涡→救赎→重建
亏损后的康复路径：觉察自己在漩涡里 → 用运动/觉察临界点自救 → 重建与财富/幸福的健康关系（"丰盛公式"：财富 ≠ 幸福）。

## 5. "抄作业"正确姿势（散户跟车指南）

1. **AI 时代研究的稀缺点在"判断"**：信息收集、框架分析已被 AI 抹平，第三层判断才值钱——别只买信息，要练判断。
2. **抄动机 > 抄动作**：搞懂"他为什么买/卖"，而不是机械跟随买卖点。
3. **跟踪选手 + 交叉验证**：多信源比对，拒绝单点信仰。
4. **避雷**：警惕只报喜不报忧的"作业"。
5. **心态**：带**复盘/分析/总结**心态，而非抄答案心态——目的是提升自己的商业认知与未来胜率。

## 6. 一页速查（决策清单）

```
① 归位：变化价值 还是 价值锚？（别混用）
② 浩哥四标准：超低估值 + 高速增长 + 巨大空间 + 异类？
   星辰三件套：便宜锚 + 向上期权 + 向下保护(分红回购≥5%)？
③ 共振：个股逻辑 × 板块 × 大盘 同向吗？
④ 仓位：真懂才重仓，没机会就拿现金；龙头可重、非主流轻仓。
⑤ 卖出触发：逻辑破坏 or 空间/估值兑现 —— 不看价格锚。
⑥ 心理体检：踏空还是被套？上头就先做 54321，再决策。
⑦ 跟车：抄动机不抄动作，交叉验证，避只报喜的雷。
⑧ 永远附风险提示，独立判断、自负盈亏。
```
