# 《三点下班》播客研究档案

> 一档以 A 股 / 港股投资为核心的"立场对撞型"陪伴播客。口号：**"股市三点收盘，我们三点下班。"**
> 本档案用于把节目内容沉淀为可复用的纪要与投资方法论，并蒸馏成可调用的分析 Skill。

---

## 一、这是什么播客

- **类型**：投资陪伴型播客，每期围绕股市行情、个股、产业趋势与股民心态展开。
- **最大特色**：两位主播**投资立场极致对立**，全程互怼、互相拷问，逼出彼此论点的漏洞，让听众在对撞中自己做判断。
- **更新状态**：已更新至**第五季（S5）**，2026 年仍在持续更新。
- **平台**：小宇宙、Apple Podcasts、蜻蜓 FM、喜马拉雅、Spotify 等 8 个平台。部分"年度展望/心理学"等重头单集为**付费节目**（约 20 元/集）。

## 二、两位主播

| | 浩哥（李永浩） | 星辰 |
|---|---|---|
| **背景** | 现职业股民，前地产从业者 | 前投资机构从业者 |
| **核心立场** | All in **AI 产业链** | All in **消费 + 恒生科技** |
| **方法论标签** | 产业革命 / **变化价值** | 均值回归 / **价值锚** |
| **典型持仓方向** | 游戏出海、AI 应用链、工业金属（铜）、精神消费、端侧芯片 | 恒生科技、消费龙头、高股息/分红回购 |
| **气质** | 进攻、赌产业拐点、容忍高波动 | 防守、看估值与安全边际、等均值回归 |

> 这套"变化价值 vs 价值锚"的对立，是理解整档节目的总纲——几乎每期都能落到这条主轴上。

## 三、季 / 内容结构（据公开单集梳理）

- **S1**：基础科普。股市"黑话"、流派（企业流/信息流/技术流）与打法（套利/周期/宏观）、从熊市学到什么。
- **S3**：方法与心理深化。投资书单、投资心理学（漩涡/救赎/重建）、十倍股巡礼。
- **S4**：2025 年度展望"钱投向哪里"、出海主题（走，出海！）。
- **S5**：2026 年度展望"我们要怎么投"、投资心理学 54321、研究价值与"抄作业"、行业冰火两重天。
